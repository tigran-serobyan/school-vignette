MSKH School Vignette

Installation

Clone GitHub repository

$ git clone git@github.com:tserobyan/school-vignette.git

You need to download and install Node.js

Then install npm packages

$ npm install

Start

Start the program whit

$ npm run start

or

$ npm run dev-start

Program is running on http://localhost

Functions

Change the username and password

Add, remove and change designs

Push

Push to GitHub.

Add your changes

$ git add .

Commit changes

$ git commit -m "description"

Push to the main branch

$ git push -u origin main
